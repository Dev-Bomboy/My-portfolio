# Personal Portfolio Website

## 📖 About the Project

This is my personal portfolio website developed using HTML and CSS. The purpose of this project is to showcase the web development skills I have learned and demonstrate my ability to create responsive and visually appealing websites.

The portfolio contains information about me, the skills I have acquired, projects I have worked on, my contact details, and links to my social media platforms.

## ✨ Features

* Responsive navigation bar
* Hero section with profile picture and introduction
* About Me section
* Skills section showcasing learned technologies
* Projects section displaying sample works
* Contact section
* Social media links
* Clean and modern user interface

## 🛠️ Technologies Used

* HTML4&5
* CSS3
* Flexbox
* Git
* GitHub
* Reponsive Design
